API HOMEPAGE
  • List Endpoints:
    • api/countries/   list of countries
    • api/events/   list of events
    • api/games/   list of games
    • api/medals/   list of medals
    • api/persons/   list of persons
    • api/sports/   list of sports
  • Detail Endpoints:
    • api/countries/<int>   details of the country
    • api/countries/<str>   details of the country
    • api/events/<int>   details of the event
    • api/games/<int>   details of the game
    • api/medals/<int>   details of the medal
    • api/persons/<int>   details of the person
    • api/sports/<int>   details of the sport
  • 2nd level Endpoints
    • api/games/<int>/events  list of events on selected games
    • api/games/<int>/events/<int>   results of selected event on selected games
    • api/countries/<int>/persons   list of sportsmen and sportswomen from selected country
  • Filter parameters
    • Olympic games filters
      • api/games/?season=summer   season: summer or winter
      • api/games/?min_year=1990   earliest year of olympic game
      • api/games/?max_year=1985   latest year of olympic game
    • Medal filters
      • api/games/?color=gold   color: gold, silver, bronze
      • api/games/?year=2016   year of competition
      • api/games/?person=stoch   part of winner's name
    • Person filters
      • api/persons/?name=cristian   part of persons name
      • api/persons/?sex=f   sex: M or F
      • api/persons/?year=1985   year of birth
      • api/persons/?min_year=1985   earliest year of birth
      • pi/persons/?max_year=1985   latest year of birth
      • api/persons/?nationality=pol   nationality, usa, pol, ger, etc.

Due to Heroku database limits, full data of only two Olympic Games is available (2016 Rio de Janeiro and 2014 Sochi). Some data might be inaccurate.
