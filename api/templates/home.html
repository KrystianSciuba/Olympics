API HOMEPAGE

  • List Endpoints:
    • api/countries/ list of countries
    • api/events/ list of events
    • api/games/ list of games
    • api/medals/ list of medals
    • api/persons/ list of persons
    • api/sports/ list of sports
  • Detail Endpoints:
    • api/countries/<int> deteils of country
    • api/countries/<str> deteils of country
    • api/events/<int> deteils of event
    • api/games/<int> deteils of game
    • api/medals/<int> deteils of medal
    • api/persons/<int> deteils of person
    • api/sports/<int> deteils of sport
  • 2nd level Endpopints
    • api/games/<int>/events list of events on selected games
    • /games/<int>/events/<int> results of selected event on selected games
    • api/countries/<int>/persons list of sportsmen and sportswomen from selected country
